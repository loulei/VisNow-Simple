Gaussian Cube Reader

The module readsvolumetric data as well as atom positions originating from the Gaussian software package.

Output data

The output field is regular 3D with affine geometry with at least one scalar component. There is a geometry object of the field.

Computation parameters

To select an input file click the browse button and select a file in the file browser or type the file name into the text input field below the button.
