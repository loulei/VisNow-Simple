Widgets - subrange slider

A float slider with two indexes determining a subrange of values.

Each index can be moved independently by the left mouse button, the whole subrange can be moved up and down by dragging the subrange bar between indexes.

If the subrange will be too narrow (of negative width or of the width less than than the extent property,

then depending on the policy property the index will stop (when policy = 0)

or will push the other index leaving the extent at the minimum value set when policy = 1.

Properly spaced legend values and ticks are displayed automatically depending on value range and widget dimensions.

The stateChanged event is generated when any index changed position, the adjusting property indicates if the subrange is actually modified.
