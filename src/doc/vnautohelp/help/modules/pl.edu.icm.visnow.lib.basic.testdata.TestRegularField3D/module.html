TEST REGULAR FIELD 3D

The module provides an example of 3D regular field datasets.

Output data

The output field are regular 3D with several scalar and vector components.

Computation parameters

The user defines the resolution value using either the slider or the text fields for minimum, maximum and current value on the right hand side of the slider, if the check box is on.

Presentation parameters

Presentation parameters are described in the common interfaces section under the Presentation Panel entry.
