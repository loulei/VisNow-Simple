MENU BAR AND TABS

Menu bar consists of

  1. Menu File
  2. Menu Edit
  3. Help menu
  4. Application Tabs Panel

Menu File contains:

  1. New –creates new Application Tab with new VisNow’s Workspace,
  2. Open data file – creates new reading module from list in Workspace Panel,
  3. Open test data – creates new test object module,
  4. Load network – opens network from saved file,
  5. Save network – saves current network into .vna file,
  6. Close – closes current Application Tab,
  7. Exit – quits VisNow.

 

Menu Edit contains:

  1. Delete – deletes selected module
  2. Colormap Editor – opens Colormap Window (see: Colormap Editor)
  3. Preferences – opens Preferences Window.

 

Preferences Window consists of Properties Tab, Favorite folders Tab and Recent Tab. Properties Tab allows to select which viewers (Viewer2D, Viewer3D, Field Viewer 3D, Orthoviewer 3D) run on startup and which viewer (Viewer 3D or Orthoviewer3D) is allowed to make automatic connections with VNGeometry ports. It also enables to set default application directory and default data directory for Application file choosers and Data file choosers.

Help menu opens help window of current module.
